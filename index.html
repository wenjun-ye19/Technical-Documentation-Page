Introduction

Python is a high-level, general-purpose programming language that is widely used in various fields such as data science, artificial intelligence, web development, game development, and many more. Python is also beginner friendly, given its code readability and English-like syntax.

Prerequisites

Python is extremely easy to learn. You will only need to know how to:

  • Use a mouse to click stuff and keyboard to type words.
  • Download software and files from the Internet.
  • Use File Explorer on Windows (a Mac/Linux equivalent will do too.)
Installing Python

Before you can start programming in Python, you will have to install it on your system. Follow these steps:

  1. Visit the Python official website
  2. Run the installer and follow the installation instructions.
  3. Make sure to check the option to add Python to your system PATH during installation.
Choosing an Integrated Development Environment

An Integrated Development Environment (IDE) can greatly enhance your programming experience. Here are a few popular options:

  • Pycharm: A powerful IDE developed by JetBrains.
  • Visual Studio Code: A versatile code editor with Python support and a wide range of extensions.
  • IDLE: Python's default IDE, simple and lightweight.

Visual Studio Code is one of the most popular choice as it is open-source, customizable and cross-platform (Able to run on Windows, Mac and Linux).

Writing Your First Python Program

Let's create a simple "Hello, World!" program to get started:

print("Hello World!")
Variables and Data Types

In Python, you can store data in variables. Python supports various data types, including:

  • int: Integer values (e.g., 5, -3, 0).
  • float: Floating-point numbers (e.g., 3.14, -0.001).
  • str: Strings of characters (e.g., "hello", 'world').
  • bool: Boolean values (True or False).

Assigning int variable: x = 5

Assigning float variable: y = 3.14

Assigning string variable: name = "Alice"

Assigning boolean variable: is_python_fun = True

Control Flow Statements
  • if...elif...else: Conditional statements.
  • for: Iterate over a sequence.
  • while: Execute a block of code repeatedly as long as a condition is true.
Functions

Functions allow you to group code into reusable blocks. They are defined using the def keyword.
